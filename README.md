## 專案簡介

## 歷史背景

1983年，中華民國對外貿易發展協會正面臨成立13年來最大的挑戰。立法院對其職能提出質疑，經濟部長趙耀東明確表達改組態度，引發各界熱烈討論。本專案整理了當時的重要報導，並加入現代觀點的分析與反思。
